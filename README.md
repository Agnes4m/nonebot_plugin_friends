## 指令

### 好友申请（权限为master_id|SUPERUSER）

- (被动) —— 接受好友请求
- 同意 —— 同意最近一次好友请求
- 同意[number] —— 同意指定qq号添加好友
- (回复消息)同意 —— 同意改申请的对象

### 群聊申请(权限为群主|管理|master_id|SUPERUSER)

- (被动) —— 接受群聊申请
- 同意[number] —— 同意指定qq号添加好友
- (回复消息)同意 —— 同意改申请的对象
